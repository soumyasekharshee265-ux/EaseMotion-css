ease-typewriter

Pure CSS typewriter animation · EaseMotion CSS submission

Basic usage
Hello, World! 👋
<span class="ease-typewriter" style="--ease-typewriter-steps: 13;"> Hello, World! 👋 </span>
Speed variants
Fast — ease-typewriter-fast Default — ease-typewriter Slow — ease-typewriter-slow
<span class="ease-typewriter ease-typewriter-fast">Fast</span> <span class="ease-typewriter">Default</span> <span class="ease-typewriter ease-typewriter-slow">Slow</span>
Cursor variants
Default blinking pipe cursor Underscore cursor variant No cursor after typing
<span class="ease-typewriter">...</span> <span class="ease-typewriter ease-typewriter-underscore">...</span> <span class="ease-typewriter ease-typewriter-no-cursor">...</span>
Staggered sequence
Line one. Line two. Line three.
<span class="ease-typewriter">Line one.</span> <span class="ease-typewriter ease-delay-700">Line two.</span> <span class="ease-typewriter ease-delay-400">Line three.</span>
